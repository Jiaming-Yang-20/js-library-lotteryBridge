Basic Information

How to play the Lottery Game

The game works like the glass bridge game in Squid Game.Players of the game first choose the prize that they want to try to win and will then be displayed the corresponding glass bridge. The glass bridge is made of tempered and regular glasses. If the user stepped on all the tempered glass and made their way to the end, they win the prize. If they stepped on regular glass, they fall and lose the prize.

What developers must specify

Developers must specify the prizes for the lottery game and the probability of winning each prize, which by default determines how many glass panels there are in the bridge. Developer can also specify the number of glass panels they want in the bridge




Examples

A Basic Bridge Game

Only the basic information is provided by the developer, e.g. the prizes for the lottery game and the probability of winning each prize. The probability of the jumper falling at each step, in this case, is 0.5. This means that one of the two parallel glass panels must break. The number of glass panels is not specified and is calculated by log0.5(prob of winning the prize). We specified the probability of winning each prize to be pretty low to show more glass panels. This means that winning this lottery game is going to be extremely hard. But this is closer to the actual squid game. Expect to lose on this one.


JS code
const ex1 = document.getElementById("ex1");
const bridgeGame1 = new Game({
data: [
{ text: "$50 gift card", chance: 10 },
{ text: "$100 gift card", chance: 5 },
{ text: "$200 gift card", chance: 1 },
],
});
const gameview1 = bridgeGame1.lottery_game();
ex1.appendChild(gameview1);



A customized Bridge Game

The basic information is provided by the developer, e.g. the prizes for the lottery game and the probability of winning each prize. The number of glass panels is also specified. This means that the probability of the jumper to fall at each step may no longer be 0.5, and is calculated such that the probability to fall at each step to the power of the number of glass panels is the prob of winning the prize. This means that both two parallel glass panels may not break. We specified the probability of winning each prize to be high this time. Expect to win the $5 prize. We also used all of the customization possible in this version. We changed the Start Screen background, the avatar of the jumper, and the color of the bridge.


JS code
const ex2 = document.getElementById("ex2");
const bridgeGame2 = new Game({
data: [
{ text: "$5 gift card", chance: 100 },
{ text: "$10 gift card", chance: 80 },
{ text: "$15 gift card", chance: 50 },
],
});
bridgeGame2.set_num_steps([4, 8, 10]);
bridgeGame2.reset_startScreenBck("bck.jpeg", "no-repeat");
bridgeGame2.change_Jumper("new_jumper.png", "new_jumper_dead.png");
bridgeGame2.change_bridge_color("#CC0000");
const gameview2 = bridgeGame2.lottery_game();
ex2.appendChild(gameview2);
