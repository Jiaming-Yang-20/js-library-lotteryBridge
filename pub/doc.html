Lottery Bridge Documentation




Getting Started

1. Load js/index.js and index.css

2. Initiate a Lottery Bridge game instance

e.g. const bridgeGame = new Game({data: [ { text: "$50 gift card", chance: 10 }, { text: "$100 gift card", chance: 5 }, { text: "$200 gift card", chance: 1 },],});

3. Call the method lottery_game() on the game instance to get a gameView

e.g. const gameview = bridgeGame.lottery_game();

4. Select the DOM element that you want to append the Lottery Bridge game to

e.g. const body = document.querySelector("body");

5. Append the gameView to the DOM element

e.g. body.appendChild(gameview);



Options

All the global options available.

Property Default Type Description
data (required) / List of objects The data for the lottery game



Objects in options.data

All the options available to customize the objects in data.

Property Default Type Description
text (required) / String The name of the prize
chance (required) / an Integer between 0 and 100 The probability of winning the prize in percentage
steps log0.5(chance) Integer The number of glass panels in the bridge. If set, the prob of failing at each jump will no longer be 0.5.



Complete configuration

const bridgeGame = new Game({
data:
[ { text: "$5 gift card", chance: 100, steps:4 },
{ text: "$10 gift card", chance: 90, steps:8 },
{ text: "$15 gift card", chance: 70, steps:10 }, ],
});




Methods

Methods on the game object to customize the lottery game.

1. lottery_game()

Explanation: Used to draw the lottery game. Return a 'div' that can be appended to a DOM element

Example:

const gameview = bridgeGame.lottery_game();

2. set_num_steps(lst_of_steps)

Explanation: Set the number of glass panels in the bridge for each prize. If set, the prob of failing at each jump will no longer be 0.5. The probability to fall at each step to the power of the number of glass panels is the prob of winning the prize.

Example:

bridgeGame.set_num_steps([4, 8, 10]);

3. reset_chance(lst_of_chances)

Explanation: Can used to reset the chances of winning each prize

Example:

bridgeGame.reset_chance([100, 90, 70])

4. reset_startScreenBck(img, repeat)

Explanation: Reset the background image of the start screen/ menu screen of the game. 'img' is in the form of String. 'repeat' indicates whether the background image should be repeated, it can either be 'repeat' or 'no-repeat'

Example:

bridgeGame.reset_startScreenBck("bck.jpeg", "no-repeat");

5. reset_gameScreenBck(img, repeat)

Explanation: Reset the background image of the game screen of the game. 'img' is in the form of String. 'repeat' indicates whether the background image should be repeated, it can either be 'repeat' or 'no-repeat'

Example:

bridgeGame.reset_gameScreenBck("bck.jpeg", "no-repeat");

6. reset_endSreenBck(img, repeat)

Explanation: Reset the background image of the ending screen of the game. 'img' is in the form of String. 'repeat' indicates whether the background image should be repeated, it can either be 'repeat' or 'no-repeat'

Example:

bridgeGame.reset_endSreenBck("bck.jpeg", "no-repeat");

7. change_Jumper(jumper_img, jumper_dead_img)

Explanation: Change the image of the jumper in the game. 'jumper_img' is the image of the jumper. 'jumper_dead_img' is the image to show when the jumper is dead

Example:

bridgeGame.change_Jumper("new_jumper.png", "new_jumper_dead.png");

7. change_bridge_color(color)

Explanation: Change the color of the bridge.

Example:

bridgeGame.change_bridge_color("#CC0000");
